# Novo Site OcupacionalMED

Este repositório contém o código-fonte do novo site da OcupacionalMED, uma empresa especializada em serviços de saúde ocupacional. Este projeto visa modernizar a presença online da empresa, oferecendo uma experiência de usuário aprimorada, um design responsivo mas mantendo as funcionalidades e dados essenciais do site atual.

## Tech Stack

- **Framework**: Next.js 15 (React) with TypeScript
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **SEO**: next-seo integration

## Design System

The design system can be found on the DESIGN.md file in this repository. It is based on Google's Stitch design system for AI. The full specification docs can be found [here](https://stitch.withgoogle.com/docs/design-md/) specification or on the repository [here](https://github.com/google-labs-code/design.md).
